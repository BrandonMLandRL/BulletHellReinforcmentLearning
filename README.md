# SimpleBulletHellReinforcmentLearning
A reinforcement learning environment and DQN network pair to train and learn inside of a custom Bullet Hell environment


** Project Setup ** 

Use Python 3.10.11

Create a virtual environment

pip install -r reqs.txt *not requirements.txt*

In the home directory run 
*pip install -e .* 
To allow for imports within the project to be resolved.
